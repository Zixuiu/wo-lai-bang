# 我来帮 - uni-app 项目

## 项目简介

「我来帮」是一个邻里互助平台的 uni-app 实现，基于 Vue 3 + uni-app 技术栈开发。

## 功能特性

- 📋 **发现需求** - 浏览附近邻居发布的帮助需求
- 📝 **发布需求** - 快速发布您需要帮助的事项
- 📦 **订单管理** - 查看您发布和接受的所有订单
- 👤 **个人中心** - 用户资料、统计数据、设置

## 技术栈

- **框架**: Vue 3 + uni-app
- **状态管理**: Vuex
- **样式**: CSS3 + 自定义样式
- **图标**: Emoji + 系统图标

## 项目结构


## 运行步骤

### 1. 安装 HBuilderX

下载地址：https://www.dcloud.io/hbuilderx.html

### 2. 导入项目

1. 打开 HBuilderX
2. 文件 -> 导入 -> 从本地目录导入
3. 选择 `wo-laibang-app` 文件夹

### 3. 运行项目

#### 方式一：内置浏览器预览
- 点击工具栏的「运行」按钮
- 选择「运行到内置浏览器」

#### 方式二：手机预览
1. 手机安装「HBuilder」调试基座（或自定义基座）
2. 使用 USB 连接手机
3. 点击「运行」->「运行到手机或模拟器」

#### 方式三：小程序预览
- 运行到微信开发者工具
- 需要配置小程序 AppID

### 4. 打包发布

#### 打包为 App
1. 发行 -> 原生 App-云打包
2. 选择 Android 或 iOS
3. 等待打包完成下载安装包

#### 打包为小程序
1. 发行 -> 小程序-微信
2. 配置小程序 AppID
3. 在微信开发者工具中上传

## 自定义配置

### 修改主题色

编辑 `pages/xxx/xxx.vue` 文件中的样式部分，修改颜色值：

```css
/* 主色调 */
--primary: #10B981;
--primary-dark: #059669;
--primary-light: #D1FAE5;
```

### 添加底部导航图标

1. 准备 4 组图标（默认和选中状态）
2. 放入 `static/tabbar/` 目录
3. 图标尺寸：48x48px，PNG 格式

## 注意事项

1. **首次运行** - 需要在 HBuilderX 中安装 Vue 3 插件
2. **真机调试** - 需要开启手机 USB 调试模式
3. **图标显示** - 如果没有添加图标文件，底部导航可能显示异常
4. **数据存储** - 使用 LocalStorage，清除缓存会丢失数据

## 常见问题

### Q: 底部导航图标不显示？
A: 需要在 `static/tabbar/` 目录下添加图标文件，参考 `readme.txt` 说明。

### Q: 如何在手机上预览？
A: 安装 HBuilder 调试基座，USB 连接手机后点击运行。

### Q: 如何打包成 APK？
A: 使用 HBuilderX 的「云打包」功能，选择 Android 平台。

### Q: 数据会保存吗？
A: 会保存在手机本地存储中，但清除应用数据会丢失。
