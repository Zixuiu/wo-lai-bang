## ✅ 验收标准

### 钱包系统
1. 模拟用户操作 → 不会影响真实用户余额
2. 充值成功 → 余额立即更新
3. 提现申请 → 余额正确扣除
4. 转账成功 → 双方余额正确变动

### 订单流程
1. 帮手标记完成 → 状态变为 pending_confirm
2. 发布者确认完成 → 余额正确扣除给帮手
3. 非相关用户 → 无法操作他人订单

### 实名认证
1. 未实名用户 → 有实名入口提示
2. 实名认证中 → 显示认证进行中状态
3. 已实名用户 → 显示实名标签

### 评价系统
1. 订单完成 → 弹出评价引导
2. 评价提交 → 评价内容正确保存
3. 评价展示 → 其他用户可见评价
