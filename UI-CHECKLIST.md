# 我来帮 App - UI修改Checklist & 验收标准

## 📋 UI设计规范总览（参考 ui-prototypes.html）

### 🎨 核心配色方案
| 用途 | 色值 | 说明 |
|------|------|------|
| **主色（Primary）** | `#10B981` | 绿色主题色 |
| **主色深** | `#059669` | 按压/激活状态 |
| **主色浅** | `#34D399` | 渐变/高亮 |
| **背景色** | `#F8FAFC` | 页面背景 |
| **卡片背景** | `#FFFFFF` | 白色卡片 |
| **文字主色** | `#1E293B` | 主要文字 |
| **文字次色** | `#64748B` / `#94A3B8` | 次要文字 |
| **成功色** | `#10B981` | 正向操作 |
| **警告色** | `#F59E0B` | 提示信息 |
| **错误色** | `#EF4444` | 错误/删除 |
| **信息色** | `#3B82F6` | 信息提示 |

### 🔤 字体规范
- **字体家族**: `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif`
- **标题字重**: 700-800 (Bold/ExtraBold)
- **正文字重**: 500-600 (Medium/SemiBold)
- **小标签字重**: 600-700 (SemiBold/Bold)

### 📐 圆角规范
| 组件类型 | 圆角值 |
|----------|--------|
| 大卡片 | `16px - 32px` |
| 中等组件 | `14px - 18px` |
| 小按钮/输入框 | `20px - 24px` (胶囊形) |
| 标签/徽章 | `50px - 100px` (完全圆角) |
| 头像 | `20px - 26px` |

### 🎭 阴影与效果
```css
/* 卡片阴影 */
box-shadow: 0 2px 12px rgba(0, 0, 0, 0.02);
box-shadow: 0 2px 16px rgba(0, 0, 0, 0.02);

/* 按钮阴影 */
box-shadow: 0 8px 20px rgba(16, 185, 129, 0.25);
box-shadow: 0 12px 28px rgba(16, 185, 129, 0.35);

/* 输入框聚焦 */
border-color: #10B981;
box-shadow: 0 6px 16px rgba(16, 185, 129, 0.05);
```

### 🔘 按钮样式
**主要按钮 (Primary)**
- 高度: `48px`
- 背景: `linear-gradient(135deg, #10B981, #059669)`
- 圆角: `24px`
- 文字: 白色, 字重800
- 阴影: `0 8px 20px rgba(16, 185, 129, 0.25)`

**次要按钮 (Secondary)**
- 高度: `48px`
- 背景: `#F3F4F6`
- 圆角: `24px`
- 文字: `#4B5563`, 字重700

### 📱 输入框样式
- 背景: `#F9FAFB`
- 圆角: `14px`
- 内边距: `10px 12px`
- 边框: `1px solid transparent`
- 聚焦时: 白色背景 + 绿色边框 + 柔和阴影
